The A / B / C Framework

Classify by intent, not opens.

A
Confirmed Intent
Clicks, replies, purchases, logins. Send with confidence.
B
Phantom (Uncertain)
Opens present but no intentional action. Handle conservatively.
C
Unengaged
No opens, clicks, or actions within a window. Eligible for re-engagement.
